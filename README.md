#  Python Utility App
This project is a collection of Python mini applications built using core programming concepts like functions, modules, and packages.
It includes utilities for:
* Math operations
* Random generators
* Date & time processing
---
## 🚀 Features
### 🔢 Math Module
* Square root, factorial, power
* GCD and LCM
* Logarithmic functions
* Trigonometry (sin, cos, tan)
* Distance calculation
---
### 🎲 Random Module
* OTP generator
* Dice simulator
* Lottery number generator
* Password generator
* Team splitter
* Random selections (student, color, etc.)
---
### 📅 Date & Time Module
* Current date & time
* Date formatting
* Days between dates
* Add/subtract days
* Day name finder
* Age calculator
---
